# Usage

```sh-session
$ npm install -g maloha
$ maloha --help
USAGE
  $ maloha [FOLDER]

OPTIONS
  -h, --help         show CLI help
  -s, --size=size    target size in kb
  -v, --version      show CLI version
  -w, --width=width  maximum width in pixels
```
